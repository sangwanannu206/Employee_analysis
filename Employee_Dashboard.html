211 missing Age values filled using per-department median. This is a safer strategy than global median as age distributions may vary by department.

✓ 211 values imputed

24 missing Salary values filled using per-department median, preserving internal pay equity logic across roles.

✓ 24 values imputed

Step 5 — Standardise Text Columns

Applied .str.strip().str.title() to Name, Status, Performance, Department, and Region columns to remove whitespace and fix casing inconsistencies.

✓ 6 columns normalised

Derived Tenure_Years from Join_Date using today's date. Provides business-ready feature without manual calculation.
